Anaconda 更新之后清理旧版本

Anaconda 更新之后可以看到 Anaconda 安装目录下面 pkgs 文件夹占了很多内存，里面是各种包的各个使用过的版本及压缩包。比如用 conda update pandas，那在 Lib\site-packages 里会更新到新版本。但 pkgs 里既有新版本，也有旧版本。推测是 conda 先把包下载到 pkgs 里，解压，然后更新到 site-packages 里正式使用。所以 pkgs 里面的内容多是无用的，为节省内存可以使用下述命令删除。

1. 赋值语句

表1：Python3 赋值语句形式
运算 解释
spam = “spam” 基本形式
spam, ham = “spam”, “ham” 元组赋值运算（位置性），,是元组关键元素。
[spam, ham] = [“spam”, “ham”] 列表赋值运算（位置性），[]是列表关键元素。
a, b, c, d = “spam” 序列赋值运算（通用性）。
a, *b = “spam” 扩展的序列解包（Python3.0）
spam = ham = “spam” 多目标赋值运算
spam += 2 增强赋值运算（相当于 spam = spam + 2）
