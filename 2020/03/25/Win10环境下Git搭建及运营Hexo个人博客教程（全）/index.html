一 准备工作

Step1 在Github上注册账号

打开Github页面，点击Sign up去注册一个账户。(Github是外国网站访问较慢，建议挂一个代理。)用户名（Username）一定要认真填写，关系到后面个人博客的访问域名。

Step2 下载安装Git

可以参考Git for Window这个页面下载Git。安装时可以自己选择安装目录，其他按照默认操作一直进行到底即可。

Step3 下载安装 Node.js

可以参考Node.js这个页面下载Node.js。安装时可以自己选择安装目录，请勾选 Add to PATH 选项。

Step4 安装 Hexo

  1. 按照Step2和Step3安装好Git和Node.js后，右键桌面空白位置，可看到 Git Bash Here 选项，点击打开；或者点击屏幕左下角开始菜单，可看到如下界面，点击打开 Git Bash Here 。(Git Bash 与 Liunx 的 Shell 相似，命令也基本一致。)
  2. 执行以下命令在自定义位置创建自己的 blog 文件夹。
